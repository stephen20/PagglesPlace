JAC Library Room Reservation System

Previous Next

I created this project from scratch with my boss while working as a web developer in the archives department at Marist College. This web based application is used internally at the school by the librarians to book rooms for students. Prior to the creation of this application the librarians were using a calendar system (similar to Google calandar) to book rooms for students, which got very messy very fast and led to mistakes being made and loads of frustration. Our application aimed to allow librarians to have access to information about each room to make sure students got everything they needed in a room, make it easy to see which rooms were reserved and which were free, make it easy to create and cancel reservations, and allow for staff to keep information on what rooms are being reserved and how each student is treating the rooms when they reserve them.
